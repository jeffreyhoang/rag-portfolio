# ResumeAI

**Summary:** A full-stack AI-powered resume builder with real-time preview, AI recommendations, and one-click PDF export.
**Tech Stack:** React, Vite, Tailwind CSS, Python, Flask, OpenAI API, WeasyPrint, python-dotenv

## What I Built

- React + Vite + Tailwind CSS frontend with a responsive real-time editing interface — changes to any section immediately update the live resume preview
- Python + Flask backend using WeasyPrint for high-fidelity PDF generation preserving professional formatting
- OpenAI API-powered recommendation engine that analyzes resume content and returns targeted suggestions for improving phrasing, highlighting impact, and tailoring content for specific roles
- All API keys managed through environment variables using python-dotenv, never hardcoded
