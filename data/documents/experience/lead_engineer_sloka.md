## About Project Sloka

An AI-powered adaptive learning platform designed for elementary school students, combining social-emotional learning with personalized academic content generation.

## What I Did

- Led end-to-end design and development of the platform, translating educational and pedagogical requirements into a scalable software architecture
- Designed and implemented the complete full-stack system using Next.js, Supabase, Tailwind CSS, and RESTful APIs
- Integrated the Gemini GPT API for AI-driven content generation, user analytics pipelines, and secure data management
- Owned technical direction from architecture decisions through deployment
- Presented Project Sloka to Cal Poly Pomona's Project Hatchery program to secure faculty mentorship, research funding, and partnerships with local schools in Pomona and Chino Hills
